What is Await in JS

* await is a new operator used to wait for a promise to resolve or reject. It can only be used inside an async function

* We can write promises in a shorter way using (async await), than using multiple then in the normal promises.

* A callback function is a function passed into another function as an argument, which is then invoked inside the outer function to complete some kind of routine or action

* Promises are used to handle asynchronous operations in JavaScript. They are easy to manage when dealing with multiple asynchronous operations where callbacks can create callback hell leading to unmanageable code
